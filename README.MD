Effect syntax:

{

    name: "unique name",
    trigger: "name of the trigger that sets off this effect"
    effect: {
        targets: [
            $
        ]
    }

}

target syntax:
1. $ = self
2. x = field slot according to x
3. !$ = everybody but self
4. !x = everybody but x
